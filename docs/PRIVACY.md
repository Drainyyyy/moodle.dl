# Privacy Policy

## Data Collection

moodle.download does **NOT** collect any personal data by default.

### Optional Telemetry (Opt-In)

If you explicitly enable telemetry, we collect:

- Number of downloaded files per session (aggregated)
- File types (e.g., pdf, zip)
- Error types (e.g., network errors)
- Extension version, browser type
- Timestamp (rounded to day)

We do **NOT** collect:

- Course IDs or URLs
- User names or emails
- IP addresses (server-side should anonymize)
- Any personally identifiable information

### Data Storage

- Download history (file hashes) is stored **locally** in your browser (`chrome.storage.local`)
- No data is uploaded to external servers unless you opt-in to telemetry

### Your Rights

- You can disable telemetry anytime
- You can reset your download history
- You can delete the extension to remove all local data
